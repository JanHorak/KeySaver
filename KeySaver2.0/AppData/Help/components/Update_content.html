KeySaver 2.0 - Hilfeseiten: Update

Bitte beachten Sie, dass bis zur finalen Version 1.0 diese Hilfeseiten unvollständig und nur auf deutsch sind

Das Updaten des KeySaver2.0 wurde bewusst einfach gehalten. Ein komplett automatisches Patchen ist angedacht, wird aber bis auf Weiteres nicht unterstützt.

Patchvorgang im Detail

  • Exportieren Sie Ihre bisherigen Daten (Hauptmenü -> Export)
  • Prüfen Sie im Update-Dialog auf eine neue Version
  • Laden Sie die Version herunter und schließen Sie den aktuell offnen KeySaver2.0
  • Löschen Sie den KeySaver2.0 (idR reicht das Löschen des Ordners)
  • Extrahieren Sie den neuen KeySaver2.0 an einen beliebigen Ort
  • Starten Sie den KeySaver2.0
  • Im Logindialog können Sie Ihre Nutzerdaten wieder importieren (Button rechts oben)
  • Selektieren Sie die vorher exportierte Zip- Datei und klicken Sie auf Import
